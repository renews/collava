# Collava

Collava is a web application to aggregate daily things you found around the web.

## Developing

To START collava environment with all dependencies `mix env.start`
To STOP environment `mix env.stop`

### Stack

* Vue
* Phoenix
* GraphQL
* Docker/DockerCompose
* Rethinkdb
